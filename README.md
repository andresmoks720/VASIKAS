## Features
- Real-time drone tracking
- Air traffic monitoring
- NOTAM (Notice to Airmen) visualization with support for:
  - Circle NOTAMs
  - Polygon NOTAMs
  - MultiPolygon NOTAMs
- Sensor coverage visualization
- Geofence management

## Mock API
This project includes an optional mock API server for development.

- **Mock API Port**: 8787 (default). Override with `MOCK_API_PORT`.
- **Run Mock API only**: `npm run dev:mock-api`
- **Run App + Mock API**: `npm run dev:all`
